select 下拉框

概述

select 下拉框是基于bootstrap的select第三方插件做的封装，可以搜索（只在客户端）、多选、分组。

其官方地址是：http://silviomoreto.github.io/bootstrap-select/

demo地址是(天朝可能会报错,因为用的jquery cdn是google的)：http://silviomoreto.github.io/bootstrap-select/examples/

github地址是：https://github.com/silviomoreto/bootstrap-select

基础用法

适用广泛的基础单选，本站点封装的组件默认是带上"selectpicker"的样式。

和普通的select用法一样，需要引入"siteSelect"插件即可。

		            				<select class="selectpicker">
	<option>Mustard</option>
	<option>Ketchup</option>
	<option>Relish</option>
</select>
<script>
	define(["siteSelect"], function () { }
</script>

分组

备选项进行分组展示

optgroup对备选项进行分组，它的label属性为分组名 

		            				<select class="selectpicker">
	<optgroup label="Picnic">
		<option>Mustard</option>
		<option>Ketchup</option>
		<option>Relish</option>
	</optgroup>
	<optgroup label="Camping">
		<option>Tent</option>
		<option>Flashlight</option>
		<option>Toilet Paper</option>
	</optgroup>
</select>

搜索

可以利用搜索功能快速查找选项，目前只是从客户端搜索。对于需要通过服务端动态搜索只能基于chang事件自己来处理。

为select设置data-live-search属性为true即可启用搜索，搜索的关键词对应的是data-tokens属性的值。

		            				<select class="selectpicker" data-live-search="true" title="select 选项" > ;
	<option data-tokens="ketchup mustard">Hot Dog, Fries and a Soda</option>
	<option data-tokens="mustard">Burger, Shake and a Smile</option>
	<option data-tokens="frosting">Sugar, Spice and all things nice</option>
</select>

限制选项的数量

多选时可以设置最大选项，对于分组的多选也可以自定义每组最大的选项。

为select设置data-max-options属性值即可设置最大选项，对于分组的选项optgroup设置data-max-options属性值表示相应的组最大选项。

		            				<select class="selectpicker" multiple data-max-options="2">
	<option>Mustard</option>
	<option>Ketchup</option>
	<option>Relish</option>
</select>
<select class="selectpicker" multiple>
	<optgroup label="Condiments" data-max-options="2">
		<option>Mustard</option>
		<option>Ketchup</option>
		<option>Relish</option>
	</optgroup>
	<optgroup label="Breads" data-max-options="2">
		<option>Plain</option>
		<option>Steamed</option>
		<option>Toasted</option>
	</optgroup>
</select>

选项选中内容格式化

选项选中时显示的是内容格式化，主要是用于多选时选择的选项太多在select框里显示不全就只显示选项的数量。

为select 设置data-selected-text-format属性'values' | 'static' | 'count' | 'count > x'。

'values'是默认值就是选中选项的值多个用“,”隔开，'static'是静态不变无论选中了多少选项，'count'当选择超过1个选项时就显示选中选项的数量，'count > x'中的x是选中的选项超过指定的数值才会显示选中选项的数量。

		            				<select class="selectpicker" multiple data-selected-text-format="count">
	<option>Mustard</option>
	<option>Ketchup</option>
	<option>Relish</option>
</select>
<select class="selectpicker" multiple data-selected-text-format="count > 2">
	<option>Mustard</option>
	<option>Ketchup</option>
	<option>Relish</option>
	<option>Onions</option>
</select>

自定义选项

可以自定义选项，比如：图标、自定义内容、副标题。

自定义选项，图标是需要设置option的data-icon的属性，自定义内容设置data-content的html内容，副标题是需要设置option的data-subtext的属性。

		            				<select class="selectpicker">
	<option data-icon="glyphicon-heart">Ketchup</option>
</select>
<select class="selectpicker" multiple>
	<option data-content="<span class='label label-success'>Relish</span>">Relish</option>
	<option data-content="<span class='label label-warning'>告警</span>">告警</option>
</select>
<select class="selectpicker" data-size="5">
	<option data-subtext="Heinz">Ketchup</option>
</select>

容器

可以选项的元素内容加入到指定的容器中。

为select 设置data-container属性，内容是指定元素的选择器，比如：body、.main-content。
