Message 全局提示

概述

Message 全局提示使用的是基于jquery的toastr第三方插件做的封装。可设置消息弹框的位置、类型、关闭事件等参数

其官方地址是：http://codeseven.github.io/toastr/

demo地址是：http://codeseven.github.io/toastr/demo.html

github地址是：https://github.com/CodeSevne/toastr

Message 基础用法

默认是从顶部中间出现，显示的动画0.3s，显示持续1s，消失动画是1s，有关闭按钮的，也可以通过点击消息框关闭(默认)或鼠标mouseout时自动关闭(默认)。

本框架是作为全局来使用的，因此在本站点内可以直接使用site.toastr来调用 Message

常规消息提示，默认背景为浅蓝色 

		            				site.toastr.info("message 消息提示!");
显示代码

不同的状态

用来显示「成功、警告、消息、错误」类型的消息提示

显示四种类型的消息提示，不同的类型适用于不同的消息提示重要性的场景。

		            				site.toastr.success("这是一条成功消息提示!");
site.toastr.warning("这是一条警告消息提示!");
site.toastr.info("这是一条普通消息提示!");
site.toastr.error("这是一条错误消息提示!");
显示代码

标题

可以添加标题。

Message的第二个参数代表的消息的标题，如果没有传值就不显示。

		            				site.toastr.success("这是一条成功消息提示，有标题哦。", "提示");
显示代码

位置

可以设置消息显示的位置。

Message连续显示时，新消息默认是接着上一个消息位置的下面显示的。如果上一条消息的位置是顶部居中且还在显示的状态那么下一条消息不管你怎么设置消息的位置都会跟上一条消息的位置保持一致。

如果必须要改变当前消息的位置得先把上一条正在显示消息移除(remove)再显示。

		            				site.toastr.info("这条消息的位置顶端右边哟", "提示", {"positionClass": "toast-top-right"});
site.toastr.info("这条消息的位置处于底部右边哟", "提示", {"positionClass": "toast-bottom-right"});
site.toastr.info("这条消息的位置处于底部左边哟", "提示", {"positionClass": "toast-bottom-left"});
site.toastr.info("这条消息的位置处于顶部左边哟", "提示", {"positionClass": "toast-top-left"});
site.toastr.info("这条消息的位置处于顶部宽度覆盖哟", "提示", {"positionClass": "toast-top-full-width"});
site.toastr.info("这条消息的位置处于底部宽度覆盖哟", "提示", {"positionClass": "toast-bottom-full-width"});
site.toastr.info("这条消息的位置处于顶部居中哟", "提示", {"positionClass": "toast-top-center"});
site.toastr.info("这条消息的位置处于底部居中哟", "提示", {"positionClass": "toast-bottom-center"});

Options

目前给出了比较重要的17个配置参数，如果还需要更多请查看toastr的官方文档。

序号 参数 说明 类型 可选值 默认值
1 closeButton 是否显示关闭图标 boolean — true
2 debug 是否使用debug模式 boolean — false
3 newestOnTop 最新出现的消息是否是显示在顶部 boolean — false
4 tapToDismiss 点击提示框是否隐藏 boolean — false
5 progressBar 是否显示进度条 boolean — true
6 onclick 点击插件回调函数 function — null
7 positionClass 弹出窗的位置 string toast-top-center、toast-top-right、toast-bottom-right、
toast-bottom-left、toast-top-left、toast-top-full-width、
toast-bottom-full-width、toast-bottom-center		 toast-top-center
8 preventDuplicates 防止消息重复,如果当前显示的信息有重复就不再去显示 boolean — true
9 showDuration 显示的动画时间(ms) number — 300
10 hideDuration 消失的动画时间(ms) number — 1000
11 timeOut 显示时间(ms) number — 1000
12 extendedTimeOut 手动点击关闭事扩展展示时间(ms) number — 1000
13 showEasing 显示时的动画缓冲方式 string swing|linear swing
14 hideEasing 消失时的动画缓冲方式 string swing|linear linear
15 showMethod 显示时的动画方式 string fadeIn|slideDown fadeIn
16 hideMethod 消失时的动画方式 string fadeOut fadeOut
17 closeOnHover 鼠标mouseout时自动关闭 boolean - true
