datetimepicker 日期时间选择

概述

flatpickr日期时间选择器支持移动手机，提供多种内置的主题效果，并且提供对中文的支持。本站点封装了本地化语言、默认配置设置、默认选择器处理。

其官方地址是：https://chmln.github.io/flatpickr

demo地址是：https://chmln.github.io/flatpickr/examples/

github地址是：https://github.com/chmln/flatpickr

基础用法

以「日」为基本单位，基础的日期选择控件，本站点封装的组件默认是带上"date-flatpickr"的样式。

引入"site-flatpickr"插件即可。

		            				<input class="form-control date-flatpickr w-small" type="text" value="" />
<script>
	define(["site-flatpickr"], function () {}
</script>

日期限制

最小日期
最大日期
可以使用的日期
禁用日期

最小日期设置属性data-min-date的值。

最大日期设置属性data-max-date的值。

可以使用的日期通过配置项：enable，它是一个数组，可以设置日期值：["2017-03-30", "2017-05-21", "2017-06-08", new Date(2017, 8, 9) ]、可以设置日期范围：[{from: "2017-04-01",to: "2017-06-01"},{from: "2017-09-01",to: "2017-12-01"}]、可以设置回调函数(返回true就启用):[function(date) {return (date.getMonth() % 2 === 0 && date.getDate() < 15);}]

禁用日期通过配置项：disable，它的值同enable一样。

注意：如果设置日期值是string类型的，必须和dateFormat相匹配。

		            				<input class="form-control date-flatpickr" type="text" value="" data-min-date="2018-02-02"/>
<input class="form-control date-flatpickr" type="text" value="" data-max-date="2017-02-02"/>
<input class="form-control" id="enableFltpickr" type="text" value=""/>
<input class="form-control" id="disableFltpickr" type="text" value=""/>
<script>
	define(["site-flatpickr"], function (siteFlatpickr) {
		siteFlatpickr("#enableFltpickr", {enable: ["2017-03-02", "2017-03-02", "2017-03-05", "2017-03-08", "2017-03-09", new Date()]});
		siteFlatpickr("#disableFltpickr", {disable: ["2017-03-02", "2017-03-02", "2017-03-05", "2017-03-08", "2017-03-09", new Date()]});
	}
</script>

设置选择范围

设置mode的值为"range"。

		            				<input class="form-control date-flatpickr" type="text" data-enable-time="true" data-mode="range" data-date-format="Y-m-d H:i"/>
<input class="form-control" id="rangeFltpickr" type="text" value=""/>
<script>
define(["site-flatpickr", "docs"], function (siteFlatpickr) {
	siteFlatpickr("#rangeFltpickr", {
		mode: "range",
		minDate: "today",
		disable: [
			function(date) {
				// disable every multiple of 8
				return !(date.getDate() % 8);
			}
		]
	});
});
</script>
